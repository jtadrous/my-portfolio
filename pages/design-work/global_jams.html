Global Jams

PHP Web Application, 2019

A full database-driven Web Application developed with PHP and MySQL. Global Jams is a website that provides users with a taste of what the top 2019 albums are from different countries around the world. The select top albums are viewable as well as purchasable right away. The site features album images, an inventory database, a login page, a search bar, and a working shopping cart. In addition, there are admin features such as adding, deleting, and editing albums in the inventory.
