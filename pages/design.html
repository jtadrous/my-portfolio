Professor's Wheel

Trivia App Project, 2018

A full web application created using HTML, CSS, and JavaScript. For high school and college students who have lower test scores and graduation rates, Professor's Wheel is a trivia app and learning activity that will make learning more interactive and fun. It features a spinning category wheel, in-app audio and buzzers, a timer, and a counter point system.

Bayside Beat

Blog/Journal Website, 2018

A full website created using HTML, CSS, and JavaScript. Bayside Beat is a blog website about the local food and recreational places to visit in the Bay area. It features images, reviews, and a column section.


Julie's Skincare Store

Vue.js Application, 2020

A single-page application created in Codesandbox using primarily Vue.js and Vuex programming. This site showcases several different types of skincare products and their details in an aesthetic display. It features images, json data files, and greensock animations.


Red Line App

HCI Final Course Project, 2019

A wireframe of an application using Mockflow.com. The Red Line is a new bus system that runs from downtown Indy to the University of Indianapolis. This fictional app was designed to enhance the user experience, allow them to search points of interests, and buy tickets. It features a map, search bar, info cards with star reviews, and multiple payment methods.
