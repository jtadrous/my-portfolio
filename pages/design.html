Keurig Website Re-creation

N315 Final Project, 2021

A re-creation of the Keurig e-commerce site that sells coffee makers and allows users to sign up with their first name, last name, email, and password. They can also login and logout easily based on user feedback. Once a user is logged in, they are able to add multiple items to their cart, remove items from their cart, and checkout. This application is fully responsive and uses various tools such as Firebase.

Professor's Wheel

Trivia App Project, 2018

A full web application created using HTML, CSS, and JavaScript. For high school and college students who have lower test scores and graduation rates, Professor's Wheel is a trivia app and learning activity that will make learning more interactive and fun. It features a spinning category wheel, in-app audio and buzzers, a timer, and a counter point system.

Global Jams

PHP Web Application, 2019

A full database-driven Web Application developed with PHP and MySQL. Global Jams is a website that provides users with a taste of the top 2019 albums from countries around the world. The site features an inventory database, a login page, a search bar, and a working shopping cart. There are also admin features such as adding, deleting, and editing albums in the inventory.

Tic Tac Toe

Board Game Project, 2020

A full web application created using HTML, CSS, and JavaScript. In this game, players alternate placing Xs and Os on the game board until either oppent has three in a row or all nine squares are filled. It features greensock animations, SVG's, several classes, and a counter-point system.

Bayside Beat

Blog/Journal Website, 2018

A full website created using HTML, CSS, and JavaScript. Bayside Beat is a blog website about the local food and recreational places to visit in the Bay area. It features images, reviews, and a column section.

Julie's Skincare Store

Vue.js Application, 2020

A single-page application created in Codesandbox using primarily Vue.js and Vuex programming. This site showcases several different types of skincare products and their details in an aesthetic display. It features images, json data files, and greensock animations.

Red Line App

HCI Final Course Project, 2019

A wireframe of an application using Mockflow.com. The Red Line is a new bus system that runs from downtown Indy to the University of Indianapolis. This fictional app was designed to enhance the user experience, allow them to search points of interests, and buy tickets. It features a map, search bar, info cards with star reviews, and multiple payment methods.

Plan of Study

HTML5 Drag-and-Drop Application, 2018

A web application created using HTML, CSS, and JavaScript. The Personalized Plan of Study is a website that allows Informatics students to pick and choose between courses and visually plan out their fours years of college. It features course boxes, color-coded categories, text input boxes, and HTML5 Drag-and-Drop. It also has 'Save', 'Reset', and 'Print' buttons at the bottom of the page.

Breakout Game

Intermediate Media Final Project, 2019

A full web game created using p5 and JavaScript. In this game, players use the left and right keyboard keys to move a paddle in the right place for a moving ball to bounce off of and hit a colored block. Each colored block hit earns them a point. It features various variables, functions, and if/else statements.
